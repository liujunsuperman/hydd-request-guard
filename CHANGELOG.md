# 更新日志

本文件记录 `@hydd/request-guard` 的重要变更。版本遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

> 说明：以下条目由提交历史整理而来，措辞以"对使用者有意义"为准，并非逐条提交的原文。

## [0.5.4] - 2026-06-30

### 优化
- 完善 npm 包元信息：补充仓库、Issues 反馈入口与官网（[happydidi.cn](https://happydidi.cn)）链接

- 文档完善与细节调整
